# tenacITy — tenacity.it

Personal website for tenacITy IT services, Baltimore MD.

## Stack
- Pure HTML/CSS/JS — no build step, no dependencies
- Hosted on Cloudways via Git deployment
- Fonts loaded from Google Fonts (Playfair Display + DM Sans)

## Deployment
This repo is connected to Cloudways for automatic deployment.
Push to `main` → Cloudways pulls and deploys automatically.

## To deploy to Cloudways
1. Push changes to `main` branch
2. In Cloudways: Application → Deployment via Git → Deploy
